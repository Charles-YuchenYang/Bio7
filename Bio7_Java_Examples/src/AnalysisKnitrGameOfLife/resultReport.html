Bio7 1.7 Knitr Example

This example demonstrates the use of Knitr inside a flow to generate a simulation report. 

Images from ImageJ can be transferred, too. 

        opts_chunk$set(dpi=72, fig.width=7, fig.height=7)

        for(i in 1:length(matrixList)){
        image(matrixList[[i]],useRaster=TRUE, col = gray.colors(2))
        title(paste("Iterations: ",i*10))
        }
